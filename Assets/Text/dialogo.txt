Bem vamos lá 
O comando if é usado em programação para criar estruturas de decisão. Estruturas de decisão são estruturas no código que escolherão, se uma condição for verdadeira ou falsa.
Por exemplo:
SE estiver chovendo eu usarei um guarda-chuva..
SENÃO eu não usarei o guarda-chuva
Agora convertendo isso para linguagem de programação seria:
If (chuva == true)
{
usar guarda-chuva
}
else 
{
não usar guarda-chuva
}
Ou seja, o programa verificará se um valor é verdadeiro ou não.

Bem vou explicar.
O comando for é usado em programação para criar estruturas de repetição. 
Estruturas de repetição são estruturas no código que repetirão alguma ação até uma quantidade de vezes pré determinada.
Por exemplo:
PARA uma determinada ação, quero que repita n vezes
AÇÃO
Agora convertendo isso para linguagem de programação seria:
for (i=0; i<100;i++)
{
printf (i);
}
Ou seja, o programa repetira a ação até 99. Quando o valor chegar a 100 ele para a execução da linha de codigo
No caso ele a variavel i começara em 0 (i=0), será verificado o valor de i é menor que 100(i<100), e por 
ultimo ele acrescenta mais 1 ao i (i++).
Ou seja repita a ação até pegar todos os itens, para sair.
